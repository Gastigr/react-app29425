## Installation
### 1 - clonar el repositorio
 > git clone https://github.com/GastonRGF/react-app29425/tree/EntregaFInal
 
 ### 2 - inicializar npm
> npm init -y
### 3 - instalar librerías y dependencias
> npm install
### 3 - correr el proyecto en un puerto local
> npm start
